# Background Music Setup

## How to add your MP3 file:

1. Add your MP3 file to this `src/assets/` folder
2. Rename it to `background-music.mp3` or update the path in `App.jsx`
3. The music will automatically play when the app loads (if browser allows autoplay)

## Current setup:
- Volume: 30% (adjustable in App.jsx)
- Loop: Enabled
- Controls: Play/Pause and Mute buttons in top-right corner

## Supported formats:
- MP3 (recommended)
- WAV
- OGG

## Note:
Modern browsers may block autoplay. Users might need to interact with the page first.
